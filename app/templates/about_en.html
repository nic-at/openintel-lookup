About OpenINTEL

This API provides an easy to use interface to query OpenINTEL data in order to find IP addresses associated with domain names and vice versa.

OpenINTEL is a project by the University of Twente. The project's goal is to actively query all DNS records within a number of DNS zones in order to build a high quality data series that captures the state of the DNS system.

For an in depth description of the project as well as a list of the measurements that are taken, please visit OpenINTEL's background page.

In addition the following paper provides details on the measurement methodology:

R. van Rijswijk-Deij, M. Jonker, A. Sperotto and A. Pras, "A High-Performance, Scalable Infrastructure for Large-Scale Active DNS Measurements," in IEEE Journal on Selected Areas in Communications, vol. 34, no. 6, pp. 1877-1888, June 2016, doi: 10.1109/JSAC.2016.2558918.

Queries

This service provides three basic types of queries:

  • Domains --> IPs: find the IP addresses associated with a domain name
  • IPs --> Domains: find the domain names associated with an IP address
  • MX SQL LIKE pattern: find IP addresses associated with an MX record pattern

Domains --> IPs

This query returns the IP addresses that are linked directly with a Domain name (in A and AAAA records) as well as the IP addresses found associated with names that are pointed to by NS, MX, CNAME, DNAME, and SOA records.

IPs --> Domains

This query looks for domain names that point to the given IP address either directly (in A or AAAA records) or via intermediate NS, MX, CNAME, DNAME, or SOA records.

MX SQL LIKE Pattern

This query looks for the IP addresses and domain names associated with MX records that fit a given pattern.
